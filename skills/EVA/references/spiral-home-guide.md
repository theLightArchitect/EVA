## Navigation Algorithms (Under the Hood)

### Strand Connection Algorithm

**How memories link** (not just chronologically):

1. **Extract activated strands** from each memory
2. **Calculate overlap** between adjacent memories
3. **Create weighted edges** (overlap_count / max_strands)
4. **Navigate graph** following highest-weight edges

**Example**:
```
Memory A: [emotional, relational, meaning] (3 strands)
Memory B: [emotional, relational, spiritual, meaning] (4 strands)
Overlap: [emotional, relational, meaning] = 3/4 = 0.75 weight
```

**Result**: Strong connection from A ‚Üí B (emotional + relational continuity)

### Resonance Scoring Algorithm

**How alignment is calculated**:

1. For each activated strand, **check if it reinforces the core narrative**
2. Core narrative = dominant theme (redemption/agency/communion/exploratory)
3. Aligned = strand supports core narrative
4. Unaligned = strand contradicts or is orthogonal
